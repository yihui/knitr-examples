## Other formatR options

We can set **formatR** options globally:


```r
options(formatR.blank = FALSE)
```


```r
1 + 1
for (i in 1:10) {
    # nothing before 10
    if (i >= 10) 
        print(i)
}
# two blank lines below
dnorm(0)
```

Or locally in one chunk via `tidy.opts`. Do not keep comments:


```r
1 + 1
for (i in 1:10) {
    if (i >= 10) 
        print(i)
}
dnorm(0)
```

Move left braces to the next line:


```r
for (i in 1:10)
{
    # nothing before 10
    if (i >= 10) 
        print(i)
}
```

```
## [1] 10
```

Indent by 2 spaces:


```r
1 + 1
for (i in 1:10) {
  # nothing before 10
  if (i >= 10) 
    print(i)
}
# two blank lines below
dnorm(0)
```

See <http://yihui.org/formatR> for details.

No warnings:


```r

```


```r
warning("no no no")
```

Select warnings using numeric indices:


```r
1:2 + 1:3
```

```
## [1] 2 4 4
```

```r
warning("no no no")
```

```
## Warning: no no no
```

Invalid indices will select nothing:


```r
1:2 + 1:3
```

```
## [1] 2 4 4
```

```r

```
